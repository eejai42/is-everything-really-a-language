What This Substrate Does

The OWL substrate generates semantic web artifacts: an OWL ontology (TBox), instance data (ABox), and SHACL-SPARQL rules for calculated fields.

Orchestration Pattern

  1. Generate: inject-into-owl.py creates OWL/RDF files
  2. Ontology: ontology.owl defines classes and properties
  3. Individuals: individuals.ttl contains instance data
  4. Rules: rules.shacl.ttl defines SHACL-SPARQL calculation rules
  5. Reason: pyshacl executes rules to compute derived values
  6. Test: Compare reasoned values against expected answers

Key Features

  • Semantic Web Standards: OWL 2, RDF, SHACL, SPARQL
  • Formal Reasoning: SHACL-SPARQL rules for inference
  • Open World: Semantic web reasoning semantics
  • Interoperability: Compatible with any RDF tooling

# === Shape with rules for Customers ===
erb:CustomersShape a sh:NodeShape ;
    sh:targetClass erb:Customers ;
    sh:rule [
        a sh:SPARQLRule ;
        rdfs:label "rule_Customers_FullName" ;
        sh:prefixes erb: ;
        sh:construct """
            PREFIX erb: <http://example.org/erb#>
            PREFIX xsd: <http://www.w3.org/2001/XMLSchema#>
            CONSTRUCT {
                $this erb:fullName ?_result .
            }
            WHERE {
    $this a erb:Customers .
    OPTIONAL { $this erb:firstName ?first_name . }
    OPTIONAL { $this erb:lastName ?last_name . }
                BIND(CONCAT(?last_name, ", ", ?first_name) AS ?_result)
            }
        """ ;
