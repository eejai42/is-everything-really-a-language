## Classification Criteria

To qualify as a **language**, a candidate must satisfy ALL of the following:

- HasSyntax = true
- RequiresParsing = true
- HasLinearDecodingPressure = true
- StableOntologyReference = true
- CanBeHeld = false
- HasIdentity = false
- DistanceFromConcept = 2
